# Dashboards · C22 CEP Chile

Sitio publicado: [https://belmarfabian.github.io/dashboards-c22/](https://belmarfabian.github.io/dashboards-c22/)

Colección de visualizaciones interactivas del C22 (Centro de Estudios Públicos) sobre la sociedad chilena. Cada dashboard se publica como un mini-sitio independiente bajo una subcarpeta, accesible desde el portal `index.html`.

## Cómo agregar un dashboard

1. Crear subcarpeta con nombre temático (ej. `nuevo_dashboard/`).
2. Poner el HTML standalone como `nuevo_dashboard/index.html`.
3. Agregar una entrada al `index.html` raíz.
4. Commit y push: GitHub Pages lo publica automáticamente.

## Stack

HTML5 + D3.js + CSS, sin frameworks. Cada dashboard es un archivo único con datos embebidos como constantes JavaScript.
